# arma-flow

A simple power flow calculator using Newton's method.

## 1. Requirements

* A newer version of [Armadillo](http://arma.sourceforge.net/)
  * Tested on 8.400.0

* The [Options](https://mulholland.xyz/docs/options/) library

* Compiler with supports C++17

## 2. Usage

See `arma-flow --help`.
